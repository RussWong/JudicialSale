# FastAPI样例

[参考链接](https://fastapi.tiangolo.com/advanced/templates/)



## 运行步骤

###Installation Package

* pip install fastapi
* pip install jinja2
* pip install aiofiles
* pip install pydantic
* pip install uvicorn

### Run Command

```
uvicorn test:app --reload
```

然后浏览器打开127.0.0.1:8000/items/1
